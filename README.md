# 🚴 Bike Sales Analysis Dashboard

## 📊 Project Overview
This project analyzes bike sales data to understand customer purchasing behavior, income trends, and regional performance.

## 🔍 Key Insights
- Customers aged 36–45 purchase the most bikes  
- North America has the highest sales  
- Customers earning ₹40k–₹70k show the highest purchase activity  

## 🛠 Tools Used
- Microsoft Excel  
- Data Cleaning  
- Pivot Tables  
- Data Visualization
